Dieses Projekt wurde im Rahmen der Bewerbung bei Pexon Consulting erstellt.

Requirements für die Anwendung:

  -Unter der Route “/” soll eine einfache Startseite stehen (z.B. “Hello Pexonian, trage hier deine Zertifizierungen ein!”
  -Es gibt ein Input Feld, in das man Cloud Zertifizierungen eintragen kann. 
  -Unten wird eine Liste aller Cloud Zertifizierungen angezeigt, die in der Datenbank abgespeichert sind.
  -Du kannst eine beliebige Datenbank verwenden, zum einfachen Starten eignet sich eine SQLite. 
  -Schreibe Unit Tests für die verschiedenen Routes
  -Es reicht wenn die App lokal läuft, Du musst sie nicht auf AWS / Azure deployen. 
  
Anschließend soll die Anwendung in einen Docker Container gehoben werden.
  
Die Anforderungen hierfür waren ein Dockerfile und die Verwendung von docker-compose.

Dieses Projekt war das erste Mal, dass ich praktisch mit Docker oder Containerisierung und Unit Testing in Python gearbeitet habe.
Deswegen sind die Requirements dafür eventuell nur teilweise erfüllt.

Außerdem müsste die Datei docker-compose.yml richtigerweise in der Ordnerstruktur eine Hierarchie höher abgelegt werden.

Das Projekt wurde innerhalb von 2 Tagen durchgeführt inklusive dem Erlernen der notwendigen Technologien wie Flask, SQLite, Docker und nicht zuletzt HTML und CSS.
